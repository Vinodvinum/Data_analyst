Excel

  • SUM: add values.
  • IF: choose between two outcomes.
  • COUNT / COUNTA: count numeric/non-empty cells.
  • XLOOKUP: find matching data by key.
  • INDEX-MATCH: flexible lookup for older files.
  • SUMIF / COUNTIF: total or count by condition.
  • Pivot table: fast summary of large tables.

SQL

SELECT col1, COUNT(*)
FROM table
WHERE condition
GROUP BY col1
ORDER BY COUNT(*) DESC;
  • JOIN: combine tables.
  • GROUP BY: summarize rows.
  • HAVING: filter grouped results.
  • ORDER BY: sort result for ranking/reporting.
  • ROW_NUMBER / RANK: top-N and ranking tasks.

Python

import pandas as pd

df = pd.read_csv("sales.csv")
df = df.drop_duplicates()
df["date"] = pd.to_datetime(df["date"], errors="coerce")
summary = df.groupby("region", as_index=False)["sales"].sum()
  • read_csv: load data.
  • dropna / fillna: handle missing data.
  • to_datetime: convert date text to real datetime.
  • groupby: summarize by category.
  • merge: join datasets.
