Project 1: Sales Performance Drop

Problem: Revenue fell 18% last month in one region.

Dataset: Orders, customers, regions, and products.

Excel

Use pivot tables and conditional formulas to compare month-over-month performance by region and product.

SQL

Write queries for revenue trend, top products, and region-wise drop analysis.

Python

Clean the data, compute revenue by month, and identify the biggest decline drivers.

Power BI

Build a dashboard with KPI cards, trend lines, and region slicers.

Recommendation: launch a targeted campaign, check pricing, and inspect stock / conversion issues.

Project 2: Customer Churn Analysis

Problem: Many users stop using the product after signup.

Dataset: Users, activity events, subscriptions, and support tickets.

Excel

Track cohorts and retention using pivots and formulas.

SQL

Build cohort and retention queries using dates and window functions.

Python

Create churn flag, retention table, and segment analysis.

Power BI

Show retention curve, churn buckets, and user segment summary.

Recommendation: improve onboarding, activation nudges, and lifecycle communication.

Project 3: Finance Collections Dashboard

Problem: Collections are behind budget and overdue invoices are rising.

Dataset: Invoices, payments, customers, monthly budget.

Excel

Build aging summaries and budget variance tables.

SQL

Join invoices, payments, and customers to calculate collections and overdue totals.

Python

Analyze payment delays, overdue patterns, and customer segments.

Power BI

Create finance dashboard with targets, variance, overdue aging, and manager filters.

Recommendation: prioritize high-value overdue accounts and track weekly recovery progress.
