Essential Formulas

Basic Aggregation

=SUM(A1:A100) - Sum all values in range
=AVERAGE(B1:B50) - Calculate mean
=COUNT(C1:C100) - Count numeric values
=MAX(D1:D50) - Find maximum value

Text Functions

=CONCATENATE(A1, " ", B1) - Join text
=LEFT(A1, 5) - Extract first 5 characters
=FIND("x", A1) - Find position of text
=UPPER(A1) - Convert to uppercase

Lookup Functions

=VLOOKUP(A1, B:D, 3, FALSE) - Vertical lookup
=INDEX(A1:A100, MATCH(B1, B1:B100, 0)) - Powerful lookup combo
=XLOOKUP(A1, B:B, C:C) - Modern lookup (Excel 365)

Pivot Tables & Summaries

Pivot tables are the fastest way to summarize large datasets. Here's the workflow:

  1. 1. Select Data Choose your entire dataset including headers
  2. 2. Insert Pivot Table Data menu → Pivot Table → Choose location
  3. 3. Configure Fields Drag fields to Rows, Columns, Values areas
  4. 4. Apply Filters Use Report Filter area for dynamic summaries
