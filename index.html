Repository and Push Workflow

Repository

Your project repository is live at Vinodvinum/Data_analyst.

Use it as the main place to store and version the guides, notebooks, and dataset packs.

Basic Git flow

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Vinodvinum/Data_analyst.git
git push -u origin main

After the first push, future updates are just add, commit, and push. Keep the landing page as the simplest way to open the whole learning set.
