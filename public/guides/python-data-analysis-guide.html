NumPy: Numerical Computing

Arrays vs Lists

Lists are flexible Python containers. NumPy arrays are optimized for numerical computation and vectorized operations.

import numpy as np

lst = [1, 2, 3]
arr = np.array([1, 2, 3])

print(lst * 2)   # list repetition
print(arr * 2)   # element-wise multiplication

Array Operations and Broadcasting

prices = np.array([10, 20, 30])
discount = 0.1

net_prices = prices * (1 - discount)
print(net_prices)

Explanation: Broadcasting lets NumPy apply a scalar or smaller array across a larger one without loops.

Pandas: The Most Important Library

Pandas is the primary tool for structured analysis in Python. It handles tabular data, cleaning, grouping, reshaping, and exports.

Analytical Techniques

Cohort Analysis

sales["cohort_month"] = sales.groupby("user_id")["order_date"].transform("min").dt.to_period("M")

Explanation: Group users by first activity month and compare later behavior.

Retention Analysis

retention = pd.crosstab(sales["cohort_month"], sales["month"])

Explanation: Measure how many users come back in later periods.

Funnel Analysis

funnel = events.groupby("event_name")["user_id"].nunique()

Explanation: Count unique users at each stage of a conversion flow.

Feature Engineering

sales["basket_size"] = sales["quantity"] * sales["unit_price"]
sales["is_high_value"] = (sales["basket_size"] > 1000).astype(int)

Explanation: Create model-ready features from raw business data.

4) Data Cleaning Pipeline

Problem: Clean messy source data and export a final analysis table.

Dataset structure: raw sales CSV with inconsistent values

def clean_sales(df):
    df = df.drop_duplicates()
    df["order_date"] = pd.to_datetime(df["order_date"], errors="coerce")
    df["region"] = df["region"].str.strip().str.title()
    df["net_amount"] = df["quantity"] * df["unit_price"] * (1 - df["discount"])
    return df

Explanation: This is a reusable pipeline pattern for analyst handoffs.

Memory Tricks and Best Practices

Remembering Pandas Operations

  • loc = label-based selection
  • iloc = integer-based selection
  • groupby = split, apply, combine
  • merge = SQL-style join

Common Mistakes and Debugging Tips

  • Always inspect dtypes before aggregating.
  • Check shape after every merge.
  • Use df.columns to confirm names.
  • Print small samples before running full pipelines.
  • Use assert statements in reusable analysis code.

Bonus: Mastery Thinking

How Analysts Think Using Python

They think in dataframes, reusable functions, reproducible notebooks, and clean outputs that can be handed off to business partners or BI tools.

Common Industry Mistakes

  • Not checking data types before math operations.
  • Overwriting raw data instead of copying it first.
  • Using loops where vectorized pandas operations are better.
  • Ignoring outliers and missing values in business metrics.

Efficient Workflow Patterns

  • Load raw data once and save a cleaned version.
  • Write helper functions for repeated transformations.
  • Use notebooks for exploration, scripts for automation.
  • Check shapes and totals at every major step.

Senior-Analyst Habit

Do not just answer the question. Produce a reliable dataset, explain the business implication, and make the work repeatable.
